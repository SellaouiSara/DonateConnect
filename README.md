
# DonateConnect

DonateConnect is a Flutter-based mobile application designed to connect donors, beneficiaries, and charitable organizations through a simple and efficient donation platform. The application helps reduce waste by allowing users to donate items they no longer need and enables people in need to request and receive donations.


* ## Features
- User registration and login
- Individual and organization accounts
- Create donation posts
- Create request posts
- Upload images with Cloudinary
- Browse available donations and requests
- Matching system between donations and requests
- User profile management
- Admin moderation system

## Technologies Used

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* Cloudinary
  

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/donateconnect.git
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the application:

```bash
flutter run
```

## Project Goal

The goal of DonateConnect is to facilitate the donation process, encourage community solidarity, and help ensure that reusable items reach people who need them instead of being discarded.
